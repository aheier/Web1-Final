The Four Design Principles

  • Percievable
  • Operable
  • Understandable
  • Robust

The Four Basic Design Principles were followed during development of this web page to create a Percievable, Operable, Understandable and Robust web page. I kept the page percievable wheras the images on the page describe what they are if they are not loaded so it can be read if they use a screen reader. The page is opperable by having all button functions and navigation working as expected and logical to use. The Page is Understandable by keeping related information grouped together and separated out into separate pages to keep pages uncluttered. The page is also Robust by using the most supported css and html and not getting to fancy with new styles, which should still work in the future.
